# Budget Planner Pro (Vanilla)

Responsive dashboard-style budget planner using **HTML + CSS + JS** (no frameworks).

## Run

- Open `index.html` in your browser.
- If the browser blocks some features when opened as a file, use any static server:
  - VS Code: install “Live Server”, then “Open with Live Server”
  - Or Windows IIS / any local web server you already use

## Features

- Dashboard + tabs (Dashboard / Transactions / Budgets / Savings Goals)
- Add/edit/delete **transactions**, **budget categories**, and **goals**
- **Monthly filter** with insights for the selected month
- **Top spending** animated canvas chart
- **LocalStorage** persistence
- Export JSON backup
